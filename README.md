# 🏦 Loan Approval Prediction App

This is a **Machine Learning web app** that predicts whether a loan will be approved or not based on user input.

---

##  Features

* Login & Signup system
* Simple banking-style UI
* ML model for prediction (Random Forest)
* Shows approval probability
* Basic rule validation (e.g., income ≠ 0)

---

##  Tech Stack

* Python
* Streamlit
* Scikit-learn
* SQLite
* Bcrypt

---

## How to Run

```bash
pip install -r requirements.txt
streamlit run app.py
```
